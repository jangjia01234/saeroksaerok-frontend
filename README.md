# saeroksaerok-frontend

회상 요법을 통해 치매를 예방하는 서비스 '새록새록'입니다. <br />
STT, AI image generator를 활용해 음성만으로 손쉽게 추억을 기록하고 그림으로 남길 수 있습니다.<br />
새록새록 떠올린 그날의 추억을 그림으로 간직하거나 친구와 공유하며 나눠보세오. <br />
